# **App Name**: HealthLink Hub

## Core Features:

- Role-Based Authentication & Dashboard Access: Secure login page for Patients, Doctors, and Admins, redirecting to their respective dashboards with appropriate access controls.
- Doctor & Patient Account Management: Admin can create Doctor accounts, Doctors can create Patient accounts and manage patient profiles (including password changes for assigned patients), and users can edit their own profiles.
- Secure Doctor-Patient Chat: Real-time, one-to-one messaging functionality between assigned Doctors and their Patients for consultations and communication.
- Appointment Scheduling & Approval: Patients can request appointments, which their assigned Doctor can then approve, denying access to any other doctor.
- Medical Document Handling: Patients can upload scan reports or images, which the assigned Doctor can view. Doctors can also upload files or prescriptions for their patients.
- AI Consultation Aid Tool: Doctors can utilize an AI-powered tool to generate preliminary suggestions or informational guidance based on patient-provided symptoms or uploaded medical data (not for definitive diagnosis).
- Admin Oversight & Reporting: Admin dashboard for managing Doctors, resetting their passwords, and reviewing patient-submitted issue reports for moderation.

## Style Guidelines:

- Color scheme: Light theme, emphasizing professionalism and calm.
- Primary color: A deep, professional blue-violet (#4A2FB8) for calls to action and important elements, conveying trust and innovation.
- Background color: A subtle, desaturated light grey-violet (#F0F0F7), providing a clean and understated canvas.
- Accent color: A brighter medium blue (#528CC2), analogous to the primary, to highlight secondary actions and key information.
- Headline and body font: 'Inter', a modern grotesque sans-serif, chosen for its excellent readability and neutral, objective feel suitable for clear information display across all roles.
- Utilize a consistent set of modern, professional line-art icons that are clear and intuitively represent medical, chat, appointment, and document-related actions.
- Implement a clean, spacious, and responsive dashboard-style layout for each user role, ensuring easy navigation and optimal readability of medical information and patient lists on various devices.
- Incorporate subtle, functional animations for UI transitions, loading states, and chat message delivery, aiming for a smooth, unobtrusive, and polished user experience without excessive distractions.
